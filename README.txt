
Este repositório contém estudos a partir do livro 'Análise de Séries Temporais em R: curso introdutório',
organizado por Pedro Guilherme Costa Ferreira. Especificamente, escrevem-se os códigos apresentados a 
cada capítulo, tentando contextualizá-los, a título de exercitar o aprendizado. A linguagem de programação 
é a mesma utilizada no livro pelos autores: R.

Bibliografia

Análise de séries temporais em R: curso introdutório / Anna Carolina Barros... [et al.] - organização Pedro
Guilherme Costa Ferreira. 1 ed. [Reimpr.]. São Paulo: GEN | Grupo Editorial Nacional. Editora Atlas LTDA.,
FGV IBRE, 2020.
